# labwork_three
лабораторная работа номер 3, где я написал свои функции сортировки, а также реализовал функции Фибоначчи и факториала рекурсивно и нерекурсивно


## Структура проекта
```
├── factorial_fibonacci/
│   ├── factorial_recursion.py     # Вычисление факториала рекурсивно
│   ├── fibonacci_recursion.py     # Вычисление числа Фибоначчи рекурсивно
│   ├── leetcode_70(factorial).py  # Вычисление факториала нерекурсивно
│   ├── leetcode_509(fibonacci).py # Вычисление числа Фибоначчи нерекурсивно
│   └── __init__.py
│
├── sorting/
│   ├── bucket_sort.py                # сортировка на корзинах
│   ├── counting_sort.py              # сортировка подсчётом
│   ├── heap_sort.py                  # сортировка деревом
│   ├── leetcode_75_bubble_sort.py    # сортировка пузырьком
│   ├── quicksort.py                  # быстрая сортировка
│   ├── radix_sort.py                 # поразрядная сортировка
│   └── __init__.py                 
│
├── stack/
│   ├── stack.py                      # реализация стека на массивах
│   └── __init__.py
│
├── tests/
│   ├── rand_generate.py              # генерация различных массивов
│   ├── sorting_test.py               # тестирование сортировок
│   ├── stack_test.py                 # тестирования стека
│   └── __init__.py
│    
├── main.py                           # запуск всех команд
├── pyproject.toml
├── .pre-commit-config.yaml
├── .gitignore
└── README.md
```

## Допущения

1. рекурсивный факториал не считает число большее 998
2. рекурсивный Фибоначчи не вычисляет число большее 998
3. в bucket sort запрещено вводить элементы, модуль которых больше 100
